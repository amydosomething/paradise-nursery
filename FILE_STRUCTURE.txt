REDUX STATE STRUCTURE
======================

store
└── cart
    ├── items: [
    │   {
    │     id: number,
    │     name: string,
    │     price: number,
    │     image: string,
    │     quantity: number
    │   }
    │ ]
    ├── totalQuantity: number
    └── totalCost: number


REDUX ACTIONS
=============

1. addToCart(plant)
   - Adds new plant to cart
   - Sets quantity to 1
   - Updates totals

2. removeFromCart(id)
   - Removes plant from cart
   - Updates totals

3. increaseQuantity(id)
   - Increments quantity by 1
   - Updates totals

4. decreaseQuantity(id)
   - Decrements quantity by 1
   - Minimum quantity: 1
   - Updates totals
